# Setup

To begin, start by exporting a theme you created using the ThemeBuilder application.
Once you have the theme saved in your file system, we switch to an IDE of your preference.

Type in the following command to create a new Flutter project


```
flutter create my_app
```

Alternatively, you can create a new Flutter project from an IDE of your choice, which will load up the contents of the Flutter starter project for you.
